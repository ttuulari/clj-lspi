# clj-lspi

A Clojure library designed to ... well, try out model-free reinforcement learning  

Build status:
[![Circle CI](https://circleci.com/gh/ttuulari/clj-lspi.svg?style=svg)](https://circleci.com/gh/ttuulari/clj-lspi)

## Usage

# Run tests
	$ lein test 

# Watch tests
	$ lein test-refresh

# Test coverage
	$ lein cloverage
